AI-Assisted Workflows

Experimental architecture designed for agentic coding tools like Claude Code. Automated contribution workflows for faster development.

Tailwind-First

Built with Tailwind CSS for easy customization and styling. No CSS-in-JS overhead.

Accessible

Built on Radix UI primitives with accessibility patterns built-in.

49+ Components

Growing collection of components covering common UI patterns. Continuously evolving through community contributions.

TypeScript

Fully typed with TypeScript for excellent developer experience and autocomplete.

Dark Mode Support

Components support dark mode through Tailwind CSS theming.
